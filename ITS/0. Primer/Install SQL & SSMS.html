1. Install SQL/SSMS

The Software for this Course

To follow this course you will need to install the SQL Server Express database and SQL Server Management Studio.
SQL Server Management Studio is the application we will use to work with the database.
The instructions on this page will walk you through the process of downloading and installing both.

How to install SQL server express

Do not try to download the SQL software directly from the Microsoft website it takes you in loop and
does not work. I have provided a different link on page 2 use this instead.
You need to know if your PC is 32 or 64 bit so that you download the correct version to find this out.
Click on the Start button and on the Right hand menu right-click on Computer and select Properties
from the drop down menu.



In the Properties page look at System Type to see what is displays 32 or 64 bit.


Click this link to download SQL Server Express, it's a 1.2GB download
SQL Server 2014 Download Link

Advanced Services x64 (this is the 64 bit version)

These links include the SQL Server Management Studio that you will use in the course.
Click the link that matches your system
When you click on a link it starts downloading immediately to the default download location for
Windows 7 which is :
C:\Users\User_Name\Downloads
